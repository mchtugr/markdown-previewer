## Run locally

Clone the project

```bash
  git clone https://github.com/mchtugr/markdown-previewer.git
```

Go to project directory

```bash
  cd markdown-previewer
```

Install dependencies

```bash
  npm install
```

Run the app

```bash
  npm run serve
```
